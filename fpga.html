> Basys 3 FPGA Project

My lab group, consisting of three members, was tasked with creating a Rover 5 robot that was capable of 3 key functions. The Rover 5 was to avoid walls, detect the colors red, green, and blue, and recieve morse code commands. The project design proved to be fully functional. While I spent most of my time designing and coding the Verilog state-machines for wall avoidance and morse code communication, I also engineered various circuits for overcurrent protection and the morse code transmitter/reciever.
